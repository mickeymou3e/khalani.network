## Setup
### Dependecies
- Foundry
```
$ curl -L https://foundry.paradigm.xyz | bash
$ source ~/.bashrc # or open a new terminal
$ foundryup
```
### Clone
```
$ git clone --recurse-submodules https://github.com/tvl-labs/khalani-protocol.git
```
### First time build
```
$ forge install
$ forge build
```

### Regular Build
```
$ forge build
```
In order to run all test 
```
$ forge test
```
